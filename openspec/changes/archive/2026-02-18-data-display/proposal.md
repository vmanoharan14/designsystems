# Proposal: Data Display Components

## Why

Data display components present information in structured formats. This change adds 6 components (avatar, card, list, data, accordions, text) using Figma tokens.

## What Changes

- Add avatar component (user avatars)
- Add card component (content containers)
- Add list component (list views)
- Add data component (data tables)
- Add accordions component (collapsible panels)
- Add text component (text areas)

## Capabilities

### New Capabilities

- `avatar`: User avatars with sizes and colors
- `card`: Content cards with hover and selected states
- `list`: List views with selection
- `data`: Data tables with sorting
- `accordions`: Collapsible accordion panels
- `text`: Text area inputs with states

## Impact

- dist/components/: 6 new CSS files
- examples/: 6 new HTML files
- tests/: Unit and E2E tests
